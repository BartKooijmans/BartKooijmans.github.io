UML model management tool

Project in progress: A UML model management tool created in Java using JSON files to store the models.

The program itself is made in Java and use JSON files to store UML models. The goal of the project was not only to create a easier to use UML management tool that supported all UML model types but do this based on a robustly created design. This project is still in progress and is part of my final thesis for my degree.
