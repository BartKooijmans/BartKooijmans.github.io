Dynamic and responsive wenshop demo

A demo for a webshop implementing a dynamic content system and responsive design.

I was recently asked to create a demo for a webshop, that implemented a responsive design and intergated a dynamic content system that could be managed in real time.

The responsive design was implemented using mostly CSS with a little bit of JavaScript for the menus. The dynamic content system uses a combination of PHP and a MySQL database. The functionality on the website uses JavaScript as well and Ajax calls were used to pull the PHP pages giving the relevant variables.

Do note that it is a demo, so functionality is limited and not everything is optimized. The code for the project including the script to create the SQL database can be found here
