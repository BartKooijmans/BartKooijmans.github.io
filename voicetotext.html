Voice to text app

A voice, both .wav files and microphone recording, to text application in python using Google's speech recognition.

I figured it would be intersting to see how difficult it was to make a application that converted an audio recording someone speaking to actual printed text.

I decided to use python for this as the language is commonly used for data analysis and natural language applications. I used the SpeechRecognition library which in turn uses various engines, like Google's voice to text services, to create the application which turned out to be rather simple and straightforward.

The project is still in progress as I want to adjust the interface a bit and add some comfort of life features, but in the meanwhile the source code can be foundhere

I'll be adding screenshots of the interface once completed. I'll also use this project in the feature to experiment with a neural network implementation in relation to natural language processing to try and improve the accuracy of the translation.
